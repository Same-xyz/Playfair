Playfair cipher rules

  1. Generate a 5x5 Matrix: The matrix is filled with letters from the key followed by the remaining letters of the alphabet. The letter 'J' is usually omitted or merged with 'I'.

    Handling Repeated Letters: If the key contains repeated letters, they are ignored in subsequent appearances.

  2. Replace 'J' with 'I': All occurrences of the letter 'J' are replaced with 'I'.

    Divide into Digraphs:Split the plaintext into pairs of letters (digraphs). If there is an odd number of letters, add a filler letter, usually 'X'.

    Handle Repeated Letters in Digraphs: If a digraph consists of the same letter (e.g., "LL"), replace the second letter with a filler letter (e.g., "LX").

  3. Letters in the Same Row: Replace each letter with the letter immediately to its right, wrapping around to the beginning of the row if necessary.

    Letters in the Same Column: Replace each letter with the letter immediately below it, wrapping around to the top of the column if necessary.

    Letters in Different Rows and Columns: Form a rectangle with the two letters and replace each letter with the letter on the same row but in the column of the other letter.
